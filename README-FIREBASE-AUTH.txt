StarQuiz V13 Firebase Auth sürümü.

Eklendi:
- Email/Password kullanıcı kayıt ve giriş ekranı
- Kullanıcı kendi hesabını oluşturabilir
- Firestore users koleksiyonuna kullanıcı kaydı ve istatistik yazılır
- Admin panelde kullanıcı sayısı ve temel istatistikler görünür

Kurulum:
1) Firebase Console'da proje oluştur.
2) Authentication > Sign-in method > Email/Password aktif et.
3) Firestore Database oluştur.
4) firebase-config.js içindeki BURAYA alanlarını kendi Firebase config bilgilerinle değiştir.
5) STARQUIZ_ADMINS içine kendi e-posta adresini yaz.
